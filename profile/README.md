#Tupane | Plan Now. Pay Later. Together.

Welcome to the official engineering hub for **Tupane**. We are building the next generation of financial coordination for Africa, starting with a transparent, community-driven Buy Now, Pay Later (BNPL) ecosystem.

### What is Tupane?
The name **Tupane** reflects our core mission: *collaboration*. We provide a seamless bridge between merchants and consumers, offering flexible payment plans that empower growth without the traditional "debt trap" friction.

---

### 🛠 Our Tech Stack
We prioritize performance, security, and developer experience (DX).
* **Backend:** Python (Django / Ninja API) & Go for high-concurrency transaction processing.
* **Infrastructure:** AWS (S3, CloudFront, Lambda) managed via Infrastructure as Code.
* **Frontend:** React / Next.js and React Native for a unified mobile experience.
* **Database:** PostgreSQL (Transactional) & Redis (Caching).
* **CI/CD:** GitHub Actions for automated, zero-downtime deployments.

---

### Security & Compliance
As a fintech, trust is our product. 
- **PCI-DSS Compliant** workflows.
- **AES-256 Encryption** for all sensitive data at rest.
- **SOC2** automated monitoring via Vanta.
- **Strict KYC/AML** integration for secure onboarding.
